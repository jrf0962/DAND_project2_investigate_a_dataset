Introduction¶

This report is a summary of the analysis of TMDb movie data set. The data set contains information about 10,000 movies collected since 1960 to 2015. The data includes informations such as budget, revenue, genre, cast, director, user ratings and popularity. In the investigation of this data set, I will be focusing to find answers to two research questions. First question is what is the most popular movie genre in each year from 1956 to 2015. Many of the movies in this data set are multi genres, which are given in a single column. Therefore in order to answer my first question, I will develop methods to consider every single genre associate with each movie separately.
My second question is what kind of properties are associated with movie that have the highest profit.
The methods that is going to develop in this project will be based on well-known and popular python packages: pandas, numpy, seaborn and matplotlib (pyplot).

Data Cleaning¶

There are 21 columns and 10866 rows in this data set. Some of these columns have large amount of non-replaceable missing values. Luckily, these columns are irrelevant for this study. There are some columns such as id and overview that are irrelevant for the study despite having complete data.
In addition, it is more reasonable to use adjusted budget and revenue for the investigation than revenue and budget. The adjusted columns (budget_adj and revenue_adj) show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time. Due to those reasons the following set of columns will be removed from the data frame.

List of columns to remove:

  • id Irrelevant
  • imdb_id Irrelevant
  • budget Use budget_adj instead
  • revenue Use revenue_adj instead
  • cast - Irrelevant
  • homepage Irrelevant and 7930 missing data (Non-replaceable)
  • director Irrelevant
  • tagline Irrelevant and over 2000 missing data (Non-replaceable)
  • keywords Irrelevant and over 1500 missing data (Non-replaceable)
  • overview Irrelevant
  • production_companies Irrelevant and over 1000 missing data (Non-replaceable)

# Separate strings between '|' 
# Return a stacked series

def separate(data):
    return data.str[0:].str.split('|', expand = True).stack()

Conclusions¶

The movie data was cleaned and used to investigate two research problems.
It was able to find that adventure and animation genres are the two most popular genres since 1960 to 2015.
The fantasy genre has the longest steak being the most popular category for seven consecutive years from 2001 to 2007.
The average profit of movies that released during the summer and holiday seasons seems to be the highest.
Action, Adventure, Animation, Family, Fantasy and Science fiction genres seems to have higher revenues.

Limitations¶

Even though the original data contained over 10000 movie data, the cleaning bring that down to about 3500 movies. There were lots of null values which were not possible to replace. There were also zero values in some data which can not be zero. For example, revenue, budget and runtime. If we can fix them the analysis could be more reasonable.
